# Master Prompt – Revision (Screenwriting Web App) **Project Name:** Revision **Type:** Web application (responsive, desktop-first) **Goal:** An alternative to Final Draft, built as a modern, collaborative screenwriting tool with real‑time co‑writing, revision tracking, and professional formatting. ### 1. Brand & Visual Identity - **App Name:** Revision - **Tagline:** *Write. Collaborate. Revise.* - **Color Palette:** - Primary: deep indigo (#1E2A5E) – for headers, buttons, accents - Secondary: warm grey (#F5F3F0) – background for script editor - Accent: muted gold (#C9A87C) – for revision highlights, active states - Success green (#2B7A4B) – for saved/online status - Error red (#D9534F) – for conflicts/deletion - **Typography:** - Body: system fonts (SF Pro / Inter) - Script editor: Courier Prime (monospace, 12pt) - **Dark mode support** – include a toggle. ### 2. Page Structure & User Flow #### A. Landing Page - Hero section with “Revision” logo, tagline, and call‑to‑action (“Start Writing Free”). - Visual: a stylised script page with cursor blinking. - Feature highlights (cards): auto‑formatting, real‑time collaboration, revision history, Beat Board. - Footer with links (pricing, about, support). #### B. Sign Up / Sign In - Clean modal or full page. #### C. Dashboard / Projects View - Grid of project cards showing script title, last edited, collaborators. #### D. Script Editor (Main Screen) - Top Toolbar, Left Sidebar (Navigator), Central Editor, Right Sidebar (Revision Panel/Comments). #### E. Beat Board / Outline View - Corkboard layout with draggable cards. #### F. Revision History - Timeline slider showing saved versions. #### G. Collaboration & Sharing Modal #### H. Settings / Preferences
